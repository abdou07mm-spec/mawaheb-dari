# مواهب داري - منصة شباب سطيف الذهبي 🌟

منصة تفاعلية لربط الشباب المتطوع بدور الشباب والمساجد في ولاية سطيف.

## ✨ المميزات
- تسجيل دخول وإنشاء حساب للمتطوعين والمدراء
- عرض الفعاليات والأحداث القريبة
- نظام نقاط وتحديات
- لوحة تنافس للمتطوعين
- معرض صور للفعاليات
- إشعارات فورية

## 🚀 تقنيات المشروع
- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## 📱 كيفية الاستخدام
1. اختر نوع المستخدم (متطوع أو مدير)
2. سجل دخول أو أنشئ حساب جديد
3. استكشف الفعاليات القريبة منك
4. تطوع وساهم في الأنشطة
5. اكسب النقاط وتنافس مع الآخرين
